# recipe-app

## Project setup
```
npm install
```

### Compiles and hot-reloads for development of Vue.js
```
npm run serve
```

### Runs the previous command, plus `nodemon server.js`
```
npm run start
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
